TabRec settings

Tab actions recommender based on your browser usage

1. Experience and other extensions

In select box below, choose the option you think describes the best your browser usage experience.
Check the following box if you are already using some other extensions for managing browser tabs. If you don't, leave the box unchecked.

2. Default recommendation mode

  • The Interactive mode means, you will always have to accept or reject recommended action.
  • Automatic mode enables us to execute the recommended action without your approval. When working correctly, this options provides the best user-experience.
We recommend to set the interactive mode first and later (when more data about your browser usage will be available for this extension) use the automatic mode.
