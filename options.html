TabRec settings

Tab actions recommender for Google Chrome

1. Choose Your experience level.

To provide the most accurate recommendation, we need to know how you use the browser. In select box below, select the option which You think describes the most Your experience in browser usage.
Please, check the following box if You are already using some other extensions for managing browser tabs. If You don't use other extensions, leave the box unchecked.

2. Choose the default recommendation mode.

  • The Interactive mode means that You will always have to choose if You want or do not want to execute recommended action.
  • The Semi interactive mode will give You a chance to stop the execution of recommendation (for about 2 seconds). Otherwise it will execute the recommendation.
  • Aggressive mode enables us to execute the recommended action without Your approval. This options provides best user-experience when working correctly.
We recommend to set the Semi-interactive mode first and later (when more data about Your browser usage will be available to this extension) use the aggressive mode.
